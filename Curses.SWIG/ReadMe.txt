========================================================================
    DYNAMIC LINK LIBRARY : Curses.SWIG Project Overview
========================================================================
This project builds the _curses.dll native Windows library containing 
the SWIG C# interface defined by pdcurses.i

/////////////////////////////////////////////////////////////////////////////

Other notes:

You need to have the SWIG executable on your PATH for the interface files to be autobuilt
and autocompied to the other project folders.
